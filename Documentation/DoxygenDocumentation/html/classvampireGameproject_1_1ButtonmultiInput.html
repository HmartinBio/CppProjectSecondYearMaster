Detailed Description

Implementation of the Class ButtonmultiInput

ButtonmultiInput class is designed such as this one contains address of MultiInputs object appended by the ButtonmultiInput class and this class allows to count points attributed by the user on each MultiInput First ComboBoxText contained in this class

Detailed Description

Implementation of the Class ButtonmultiInputOneComboBox

ButtonmultiInputOneComboBox class is designed such as this one contains address of MultiInputOneComboBox object appended by the ButtonmultiInput class and this class allows to count points attributed by the user on each MultiInputOneComboBox ComboBoxText contained in this class
